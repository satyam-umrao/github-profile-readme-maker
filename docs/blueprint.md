# **App Name**: Git Profile Pro

## Core Features:

- Profile Details Input: Collects user's name, role, bio, location, education, and work information.
- Tech Stack Selection: Enables users to select technologies from a predefined list to showcase their tech stack.
- Social Links Integration: Allows users to input links to their various social media profiles.
- README Generation: Generates a markdown README file based on the provided profile details, tech stack, and social links.
- AI Profile Enhancement: Suggests improvements to user bios and profile taglines using a generative AI tool.
- Profile stats: Tracks the README size in lines.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), representing professionalism and trustworthiness.
- Background color: Very light grey (#F5F5F5) for a clean and modern feel.
- Accent color: A warm yellow (#FFEB3B) to highlight key elements and calls to action, analogous to indigo.
- Body and headline font: 'Inter' for a clean and versatile design. 
- Code font: 'Source Code Pro' for displaying code snippets and the README preview.
- Use a consistent set of minimalist icons to represent different sections and technologies.
- Divide the page into clear sections using panels and dividers, maintaining a balanced and responsive layout.
